Get Avatar Image
Use GetPlayerAvatarTexture((Texture2D) => {})to get the current player's Texture2D object.
galaxyController.GetPlayerAvatarTexture((imageTexture) => {
avatarImage.sprite = Sprite.Create(imageTexture, new Rect(0.0f, 0.0f, imageTexture.width, imageTexture.height), Vector2.one * 0.5f);
});
Public URL: All player avatars are accessible at the public URL: https://galaxy.us/api/v1/{global_player_id}/avatar.png
