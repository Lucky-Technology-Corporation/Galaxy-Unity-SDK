Show Exchange UI
Recommended!
No Backend Required: Showing the UI will present a clean interface for the user to charge points and refer friends to your game.
Before you show the payment screen, you need to subscribe to the didBuyCurrency(amount)
void Start(){
galaxyController.didBuyCurrency += rewardPlayerWithCurrency;
}
