Show Exchange UI
Recommended!
No Backend Required: Showing the UI will present a clean interface for the user to charge points and refer friends to your game.
Before you show the payment screen, you need to subscribe to the didBuyCurrency(amount)
void Start(){
galaxyController.didBuyCurrency += rewardPlayerWithCurrency;
}
​
private void rewardPlayerWithCurrency(int amount){
playerGems += amount;
}
​
Then, use ShowPayment(optional_left_margin?, optional_top_margin?, optional_right_margin?, optional_bottom_margin?) to show the payment screen.
The screen will automatically hide on successful point transfer or referral.
//Show UI for all leaderboards full screen
galaxyController.ShowPayment();
​
//Show UI with 100px of padding at the top
//You must pass a leaderboard ID to use the margins
galaxyController.ShowPayment(0, 100);
​
//Optionally hide the screen
galaxyController.Hide();
