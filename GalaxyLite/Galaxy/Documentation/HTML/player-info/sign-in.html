Sign In
This is optional, as an anonymous user account is created for every player as soon as they start the game. You can submit scores, show the leaderboard, etc without signing the user in. The leaderboard may prompt the user to sign in at sensible points (e.g. saving an avatar).
Call SignIn() to prompt the user to enter their phone number and sync their contacts if they haven't already. You do not need to call sign in before using the other methods.
Use this when you'd like to control the flow yourself. This creates a full screen web view which is optionally removed on successful sign in. If shouldCloseOnCompletion is set to false, the leaderboard will be shown on successful sign in.
galaxyController.SignIn(bool shouldCloseOnCompletion); //Will create a full screen webview
Delegate
Assign the didSignIn(string playerId) delegate to listen for success/failure. Player ID is null if the sign in was unsuccessful
void Start(){
galaxyController.didSignIn += myPlayerDidSignInFunction;
}
