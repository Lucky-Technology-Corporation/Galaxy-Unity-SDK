Other Player Info

Basic Info

Use GetPlayerInfo((PlayerInfo) => {}) to get the current player's basic information.
galaxyController.GetPlayerInfo((playerInfo) => {
Debug.Log("This player is named " + playerInfo.nickname);
});
This returns a PlayerInfo object
public class PlayerInfo{
public bool is_anonymous; //false if the player has signed in with their phone
public string global_id; //player's global id
public string id; //player's ID
public string? nickname; //null if unauthenticated
public string? profile_image_url; //null if unauthenticated
}
​
Delegate
Assign the infoDidChange(PlayerInfo playerInfo) delegate to listen for success/failure. Player ID is null if the sign in was unsuccessful
void Start(){
galaxyController.infoDidChange += myPlayerInfoUpdateFunction;
}
​
private void myPlayerInfoUpdateFunction(PlayerInfo info){
Debug.Log("New nickname is " + info.nickname);
}

Friends

Use GetPlayerFriends((List<PlayerInfo>) => {}) to get a list of players that the user is friends with. This list may be empty, and will be if the user hasn't signed in.
galaxyLeaderboard.GetPlayerFriends((friendList) => {
//Print this user's friends' nicknames
for(var i = 0; i < friendList.Count; i++){
Debug.Log(friendList[i].nickname + "\n");
}
});

Rankings

If you've been submitting scores, use GetPlayerRecord(leaderboard_id, (PlayerRecord) => {}) to get the current player's ranking information.
galaxyLeaderboard.GetPlayerRecord("leaderboardId", (playerRecord) => {
Debug.Log("This player is ranked " + playerInfo.rank + " of " + playerInfo.total_players);
});
This returns a PlayerRecord object
public class PlayerRecord{
public bool is_anonymous; //false if the player has signed in with their phone
public string global_id;
public string id; //player's ID
public int highscore;
public int ranking;
public int total_players;
public string? nickname; //null if unauthenticated
public string? profile_image_url; //null if unauthenticated
}
