Player IDs

Set ID

If you want to use your own ID, set the current player ID with SetPlayerID. Calling this will overwrite the previous player ID. This is not required and does not change the global player ID.
If a user doesn't have an account, their ID will not change after they sign in. If they do have an account, their ID will update to their existing ID.
galaxyController.SetPlayerID("your_custom_id");

Get ID

You may need the player ID for multiplayer or tracking purposes. You can get the player's current ID with GetPlayerID.
string playerId = galaxyController.GetPlayerID();
You can also get the player's global ID, which stays consistent from game to game once the user has signed in. Setting the player's ID has no effect on this ID.
string globalPlayerId = galaxyController.GetGlobalPlayerID();
