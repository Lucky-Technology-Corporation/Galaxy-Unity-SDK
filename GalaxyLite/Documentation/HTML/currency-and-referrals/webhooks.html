Webhooks
Subscribe to webhooks through your dashboard to listen for events. The following event categories are available
  • charge the user was charged points (and you got paid
​

Charge Event

Event types:
charge.success
charge.failure.nsf User doesn't have enough money
charge.faliure.notfoundUser not found
charge.failure.timeoutUser didn't confirm in time
charge.failure.rejected User rejected the charge
Example Body:
{
"event_type": "charge.success",
"user_id": "001",
"charge_amount": 80, //the amount of credits that were charged
"reward_amount": 15, //the amount of your currency you defined
"product_description": "Gems",
"idempotency-key": "{uuid}", //autogenerated if not provided in the initial request
"signature": "*****" //hmac is the SHA256 hash of this payload using your secret key
}
