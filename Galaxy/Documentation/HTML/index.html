Unity Set Up

The leaderboard and avatar editor can be conveniently displayed within any Web View. For your convenience, we've bundled the amazing open source gree/unity-webview library with the Unity SDK. Galaxy controller uses this to display the leaderboard and avatar editor.
You'll interact with GalaxyController. Reference the object in whatever script you're going to use to report scores, display the leaderboard, and match players. You can reference the object in multiple scripts if you'd like.
public GalaxyController galaxyController;
Find the Galaxy folder (should be under Packages) and drop the Galaxy prefab into your scene from Galaxy/Galaxy/GalaxyController.prefab.
Select the prefab and input your SDK Key (publishable key) found in your dashboard.
Now you can reference the prefab in whatever script will be interacting with it.
